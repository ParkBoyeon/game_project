# 🎮 게임 개발 전략 (Claude Code용)

## 공통 원칙

- 단일 HTML 파일로 구현 (외부 라이브러리 없이 순수 HTML + CSS + JS)
- 모바일/PC 모두 동작하도록 반응형 처리
- Canvas API 기반으로 렌더링
- 각 게임은 독립적인 파일로 분리

---

## 🍄 슈퍼마리오 게임

### 파일명
`mario.py`

### 구현 범위
- 1개 스테이지 (고정 맵, 스크롤 사이드뷰)
- 마리오 캐릭터 이동 및 점프
- 굼바(적) 등장 및 충돌 처리
- 코인 블록, 벽돌 블록
- 버섯 아이템 (먹으면 마리오 커짐, 한 번 더 맞아도 살아남음)
- 스테이지 끝 깃발 (닿으면 스테이지 클리어)
- 굴뚝(파이프) 장식 및 진입 이벤트
- 점수 및 목숨 표시

### 핵심 기술 구조
```
- Canvas 2D API로 렌더링
- 게임 루프: requestAnimationFrame
- 충돌 감지: AABB (Axis-Aligned Bounding Box)
- 중력: 매 프레임 velocityY += gravity
- 카메라: 마리오 위치 기준 canvas 오프셋 이동
```

### 조작키
- `← →` 또는 `A D`: 이동
- `Space` 또는 `W` 또는 `↑`: 점프
- 모바일: 화면 하단 터치 버튼

### Claude Code 지시 프롬프트
```
mario.py 파일을 만들어줘.

요구사항:
1. 마리오 캐릭터 (픽셀 스타일, 빨간 모자)
2. 사이드 스크롤 고정 맵 (스테이지 시작 ~ 끝이 있는 구조)
3. 플랫폼(바닥, 벽돌 블록, 물음표 블록) 위에서 걷기 및 점프
4. 굼바 적 캐릭터 (마리오가 위에서 밟으면 처치, 옆에 닿으면 피해)
5. 아이템 시스템 (물음표 블록을 밟으면 랜덤 등장):
   - 코인: 일정 확률로 코인만 나오고 점수 추가
   - 버섯 🍄: 작은 마리오일 때 등장, 먹으면 큰 마리오로 변신
   - 파이어 플라워 🌸: 큰 마리오일 때 등장, 먹으면 파이어 마리오로 변신
   - 파이어 마리오: 스페이스바 또는 별도 키로 파이어볼 발사, 적 처치 가능
   - 피해 단계: 파이어 마리오 → 큰 마리오 → 작은 마리오 → 목숨 감소
6. 굴뚝(파이프):
   - 맵 곳곳에 초록색 파이프 배치
   - 특정 파이프 앞에서 아래 방향키 누르면 파이프 안으로 들어가는 연출
   - 파이프 안으로 들어가면 화면 전환 후 다시 나오는 이벤트
7. 스테이지 끝에 깃발 폴 배치:
   - 깃발에 닿으면 마리오가 폴을 타고 내려오는 애니메이션
   - 스테이지 클리어 화면 표시 및 점수 합산
8. 코인 수집 및 점수 표시
9. 목숨 3개, HUD에 마리오 아이콘으로 표시
10. 키보드 조작: 방향키 또는 WASD, 스페이스바 점프
11. 모바일용 터치 버튼 (왼쪽, 오른쪽, 점프)
12. 게임 오버 및 재시작 기능

픽셀아트 스타일로 꾸며줘. 배경은 하늘색, 구름과 산 장식 포함.
```

## 🟦 테트리스 게임

### 파일명
`tetris.py`

### 구현 범위
- 표준 테트리스 7종 블록 (I, O, T, S, Z, J, L)
- 블록 회전, 이동, 낙하
- 줄 완성 시 제거 및 점수 계산
- 다음 블록 미리보기
- 레벨 시스템 (속도 증가)
- 최고 점수 저장 (localStorage)

### 핵심 기술 구조
```
- Canvas 2D API로 메인 보드 렌더링
- 게임 보드: 10x20 2D 배열
- 블록: 4x4 행렬로 회전 처리
- 충돌 감지: 보드 배열 값 체크
- 자동 낙하: setInterval로 타이머 관리
- 레벨업: 10줄 제거마다 속도 증가
```

### 조작키
- `← →`: 블록 좌우 이동
- `↑` 또는 `Z`: 블록 회전
- `↓`: 빠른 낙하 (소프트 드롭)
- `Space`: 즉시 낙하 (하드 드롭)
- 모바일: 스와이프 제스처

### 점수 시스템
| 제거 줄 수 | 점수 |
|-----------|------|
| 1줄 | 100 × 레벨 |
| 2줄 | 300 × 레벨 |
| 3줄 | 500 × 레벨 |
| 4줄 (테트리스) | 800 × 레벨 |

### Claude Code 지시 프롬프트
```
tetris.py 파일을 만들어줘.

요구사항:
1. 표준 테트리스 7종 블록 (I, O, T, S, Z, J, L), 각각 다른 색상
2. 10x20 게임 보드
3. 블록 이동 (좌, 우, 하), 회전, 하드 드롭(스페이스바)
4. 줄이 꽉 차면 제거 및 위 블록 내려오기
5. 다음 블록 미리보기 패널
6. 점수판: 현재 점수, 최고 점수(localStorage 저장), 레벨, 제거한 줄 수
7. 레벨 시스템: 10줄 제거마다 레벨업, 낙하 속도 증가
8. 키보드 조작: 방향키, Z키 회전, 스페이스바 하드드롭
9. 모바일 스와이프 제스처 지원
10. 게임 오버 화면 및 재시작 버튼
11. 블록 낙하 위치 미리보기 (고스트 블록, 반투명)

다크 테마로 디자인. 네온 컬러 블록으로 레트로 감성 살려줘.
```

## 📁 개발 순서 권장

```
1단계: tetris.py 먼저
   → 로직이 단순하고 검증하기 쉬움
   → Canvas 기초 익히기 좋음

2단계: mario.py
   → 물리 엔진, 카메라, 충돌 처리 등 복잡도 높음
   → 테트리스로 Canvas 감 잡은 후 진행
```
